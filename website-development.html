I built this website!

A simple, responsive digital CV website created to showcase experience, projects, and contact details using modern front-end practices.

The homepage and featured project cards were built using Tailwind CSS and a clean, responsive layout that works on desktop and mobile. The site is designed to feel modern while keeping accessibility and readability front of mind.

The goal was to create a personal website that functions as both a portfolio and a CV, with clear sections for work, employment history, education, endorsements, and contact information. It was important that the site loads quickly and works well on all screen sizes and works in both light and dark modes.

Alpine.js is used for the dark mode toggle, which respects the user's system preference and saves their choice for future visits. The design uses a simple color palette with an accent color to highlight important elements and calls to action.

Overall, this project was a fun opportunity to apply front-end development skills to create a personal website that effectively communicates my experience and projects while also demonstrating attention to detail in design and user experience.

Key features

  • Responsive layout with mobile-first design
  • Dark mode toggle
  • Accessible semantic structure

Technologies

  • HTML
  • Tailwind CSS
  • Alpine.js
